### 🎁 **프로젝트 개발 이유**

평상시에 공부할때나 일정이 있을때 투두 앱을 많이 사용하는데 기능들을 직접 구현해 보고 싶어서 개발했습니다. 

### 🔨 **기술 스택**

App : React Native

Back : firebase

백엔드는 집중적으로 개발하지 않고 프론트에 집중하기 위해 파이어베이스를 사용하여 간단하게 개발했습니다. 😊

### 🔍 **앱 기능**

- 할 일 입력하기
- 할 일 삭제하기
- 할 일 수정하기
- 할 일 완료하기
