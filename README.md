## 📱 Todo App

React Native의 기술을 익히기 위해 가장 간단하고도 일상에 필요한 Todo App을 간단하게 개발했습니다. </br>
To-Do 앱은 CRUD 기능을 구현하는 데 적합해, 전반적인 앱 구조를 학습하는 데 큰 도움이 되었습니다.

---

## ✨ 주요 기능

- Todo 입력, 저장, 수정, 삭제, 확인, 취소
- Todo 드래그로 순서 이동

---

## 🛠 기술 스택

- App : React Native
- Backend : Firebase
